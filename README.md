# Lead Capture Landing Page

A responsive landing page built to capture leads through a functional contact form integrated with EmailJS.

## 🚀 Live Demo
[View Live](#) <!-- we'll update this link after deployment -->

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox, responsive design, mobile-first)
- JavaScript (ES6+)
- EmailJS (form integration and email delivery)

## ✨ Features
- Responsive layout for mobile and desktop
- Lead capture form with frontend validation
- Real email delivery via EmailJS integration
- Success feedback message after form submission
- Clean and modern UI
